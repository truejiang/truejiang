HTML Drag and Drop API events

拖拽的意思，顾明思意是鼠标拿起一个元素，和放下一个元素的过程。
使用时需要先给元素设置属性 draggable 为 true。Drag 有以下 API events,我们先逐个认识下。

  • drag: 当元素或者选择的文本被拖动时触发
  • dragend: 当拖放事件在施放操作结束时触发（通过释放鼠标或者单击 esc 键）
  • dragenter: 当拖动的元素或者文本进入有效的放置目标时触发
  • dragleave: 当拖动的元素或者文本离开有效的放置目标时触发
  • dragover: 当元素或者的选择的文本被拖拽到一个有效的放置目标上时触发
  • dragstart: 当用户开始拖动一个元素或者一个选择文本时触发
  • drop: 当元素或者是选中的文字被拖拽释放到一个有效的释放目标时触发

和所有的 DOM 事件一样，监听的事件会返回一个 event 对象。其中包含了我们本次监听对象的一些信息。比较有用的有

  • target: 被拖动的元素下面的元素（这个视情况，当是 drag 时，代表的是被拖拽的元素）
  • type: 被监听的事件类型
  • screenX 和 screenY: 鼠标在整个屏幕中所在的坐标值
  • clientX 和 clientY: 鼠标所在 target 容器中的坐标值
  • button: 表示我们触发事件时是鼠标的哪个键位去触发的，1 表示左键，2 是中间滚轮，3 是右键，其他可以查看 more info
