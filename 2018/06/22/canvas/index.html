用canvas画一个动态的时钟

创建于 2018-06-22 | 分类于 JavaScript | 阅读人数:

这两天看了JavaScript 高级程序设计（第 3 版）的第 15 章，学习了如何使用 canvas，为了巩固知识，写了一个
可以世界时钟墙。下面我们一步步用 OOP 的设计方式，来写完这个程序吧。下面先看下演示的效果

前置知识

首先我们需要用的是 HTML5 的新功能 <canvas>元素。可以通过 JavaScript 动态的在这个元素区域绘制图形。
使用这个元素必须先设置它的 width 和 height 属性，用于指定绘图的区域大小。
想要在这个 canvas 画布上绘图，需要先取得绘图上下文。而取得绘图上下文对象的引用，则需要调用 getContext() 方法并入上下文的名字。这里我们画的平面图，
所以传入’2d’，就可以取得 2D 上下文对象。

2D 上下文对象提供了一些方法，用以绘制简单的 2D 图形，如矩形，弧线，路径等。2D 上下文通过坐标去寻找绘制的起点，这个起点从 <canvas> 元素的左上角，坐标值为(x = 0, u = 0),所有的坐标基于这个原点计算，x 值越大表示越靠右，y 值越大表示越靠下，反之亦然。

canvas 绘制时常用的填充方式有 2 种 一种是 stroke 开头，表示画轮廓，一种是 fill 开头，表示填充全部内容。比如画一个正方形，stroke 表示设置正方形边框，fill 表示整个正方形的内部。

  • 颜色设置
    • ctx.strokeStyle 设置描边颜色样式。即画画的轮廓
    • ctx.fillStyle 设置填充颜色样式。填充整个画
  • 直接画图
    • fillRect(x, y, width, height) 填充式的画一个矩形。x, y 为开始的坐标值，width 和 height 分别表示宽高
    • strokeRect(x, y, width, height) 画一个带轮廓的矩形。x,y,width,height 同上
    • clearRect(x, y, width, height) 表示在 x, y 坐标位置，清除一个宽高为 width,height 的矩形
  • 绘制路径
    • arc(x, y, radius, startAngle, endAngle, counterclockwise): 以 x,y 为圆心绘制一条弧线，弧线半径为 radius，起始和结束角度即弧度用 startAngle 和 endAngle 表示。counterclockwise 表示起止弧度是否反转，用布尔值表示。
    • lineTo(x, y) 从上一个点开始绘制一条直线，落点在 x, y 坐标。可以理解为画一条直线的终点。
    • moveTo(x, y) 讲绘图游标移动到 x, y 坐标，不画线。可以理解成你把笔提起来，落在了 x,y 新坐标开始准备画图。
    • rect(x, y, width, height) 从 x, y 开始绘制一个矩形，width 和 height 代表宽高。
      注意上述的绘制路径都只是创建路径，但实际并没有开始实际绘图，如果想绘制一条连接到路径起点的线条，可以用 closePath()。如果路径已经完成，可以使用 fill()填充，填充的样式则是调用前面的设置的 fillStyle。还可以使用 stroke() 对路径描边，描边的样式对应的是设置的 strokeStyle。
  • 绘制文本
    • fillText(textContent, x, y) textContent 表示文本内容，x,y 表示放置的位置
    • strokeText(textContent, x, y) 同上
    • 上述的 2 个文本样式通过 font, textAlign, textBaseline 去设置。
  • 变换
    • rotate(angle) 围绕原点旋转图像弧度
    • scale(scaleX, scaleY) 缩放图像，分别代表 x, y 方向缩放的比例
    • translate(x, y) 将坐标移动到(x, y)。执行后坐标(0, 0)会移动到(x, y)坐标，被起代替

其他的一些内容可以看高程三或者MDN web dos - canvas。

我们先设计画一个时钟出来，时钟分为几个部分，

  • 时钟的圆框
  • 时钟上一圈的点，表示把整个圆划分成了 60
  • 数字代表小时数，把圆分为了 12 份，同样也对应上面圆的 60 个点中的 12 个加粗的圆点
  • 时针分针秒针
