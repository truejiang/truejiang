JSON

含义

JSON 类似于对象，都用于存储一组无序的数据，但 JSON 只能存储简单值（简单值包括字符串，数值，布尔值，null，但是 JSON 不支持 undefined），对象，数组，另外还有3个区别：JSON不能使用单引号，只能用双引号；key 值也要用双引号包起来；JSON没有变量的概念，所以不需要赋值，直接写数据就行。
